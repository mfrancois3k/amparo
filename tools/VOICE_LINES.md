# Amparo officer voice — recording manifest

Record each line below with your voice tool. **Two voices**: male officer and female officer — every line once per voice.

## Rules
- **File format:** MP3 (or give me WAV/M4A and I'll convert)
- **File names must match exactly** — that's how the app finds them
- Male voice files → folder `audio/en/m/` · Female voice files → folder `audio/en/f/`
  (or just name them `m_v0_0.mp3` / `f_v0_0.mp3` and drop them all in Downloads — I'll sort them)
- **Consistency is everything:** same mic, same distance, same room, same session if possible. One inconsistent clip breaks the immersion more than a robotic one.
- Keep ~0.2s of silence at start and end. No music, no effects.

## Delivery guide
- **CALM** — professional, unhurried, almost bored. Routine stop.
- **CURT** — clipped, impatient. Short sentences hit hard. No shouting.
- **HOSTILE** — faster, lower, pressuring. Menace through pace and coldness — never screaming, never slurs.
